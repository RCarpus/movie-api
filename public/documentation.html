Movie API Documentation

Author: Ryan Carpus

This API provides users with access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

Function Endpoint URL HTTP Method Request body data format Response body data format
Get data for all movies /movies GET None A JSON object holding complete data for all movies.
Get data for a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing an ID, title, description, genre, director, image URL, and flag for is-featured.
example:
{
"_id": "6197bc855ba878fd0afa7418",
"Title": "The Lion King",
"Description": "Animated lions perform a completely faithful rendition of Hamlet.",
"Genre": {
"Name" : "animated",
"Description" : "movies made by drawing lots of pictures and then talking on top of them"
}
"Director": {
"Name" : "Rob Minkoff",
"Bio" : "Robert Ralph Minkoff is an American filmmaker. He is known for co-directing the Academy Award-winning Disney animated feature The Lion King, along with directing Stuart Little, Stuart Little 2, The Haunted Mansion, The Forbidden Kingdom, and Mr. Peabody & Sherman.",
"BirthYear" : 1962
}
"ImageUrl": "theLionKing.png",
"Featured": false;
}
Get information about a genre by name /genres/[Genre] GET None A JSON object holding a description of the requested genre.
example:
{
"Name": "horror",
"Description": "Movies that make you scared."
}
Get data about a director by name /directors/[Name] GET None A JSON object holding data about about the requested director.
example:
{
"Name": "Tetsuya Nomura"
"Bio": "Tetsuya Nomura is a Japanese video game artist, designer and director working for Square Enix. He designed characters for the Final Fantasy series, debuting with Final Fantasy VI and continuing with various later installments."
"Birth-year": 1970
}
Register a new user /users/register POST A JSON object holding data for a new user
example:
{
"Username": "johndoe"
"Password": "aStrongPasSwOOrd"
"Email": "johndoe@gmail.com",
"Birthday": "1995-08-24"
}
 A JSON object holding data for the new user.
example:
{
"_id" : "619b8cfc843f95b8e96b8e66",
"Username": "johndoe"
"Password": "aStrongPasSwOOrd"
"Email": "johndoe@gmail.com",
"Birthday": "1995-08-24"
"FavoriteMovies": []
}
Update a user's data /users/[Username] PUT A JSON object holding the data to be updated for the user
example:
{
"email" : "KILLALLHUMANS@robots.com"
} 		A JSON object holding the complete updated user data.
example:
{
"_id" : "6197ba7e5ba878fd0afa7415",
"Username" : "KillerRobot",
"Password" : "I am not a robot",
"Email" : "KILLALLHUMANS@robots.com",
"Birthday" : "1970-01-01T00:00:00.000Z",
"FavoriteMovies": [
"6197b3dd5ba878fd0afa740a",
"6197b5925ba878fd0afa740f",
]
}
Add a movie to a user's favorites list /users/[Username]/movies/[:MovieID] POST None A JSON object holding the updated user data.
example:
{
"_id" : "6197ba7e5ba878fd0afa7415",
"Username" : "KillerRobot",
"Password" : "I am not a robot",
"Email" : "KILLALLHUMANS@robots.com",
"Birthday" : "1970-01-01T00:00:00.000Z",
"FavoriteMovies": [
"6197b3dd5ba878fd0afa740a",
"6197b5925ba878fd0afa740f",
]
}
Remove a movie from a user's favorites list /users/[Username]/movies/[:MovieID] DELETE None A JSON object holding the updated user data.
example:
{
"_id" : "6197ba7e5ba878fd0afa7415",
"Username" : "KillerRobot",
"Password" : "I am not a robot",
"Email" : "KILLALLHUMANS@robots.com",
"Birthday" : "1970-01-01T00:00:00.000Z",
"FavoriteMovies": [
"6197b3dd5ba878fd0afa740a",
]
}
Delete an existing user /users/deregister/[Username] DELETE None A text message indicating that a user was deleted
example:
"KillerRobot was deleted."
Retrieve data for all users /users GET None A JSON object holding all data for all users.
Retrieve data for a specific user /users/[Username] GET None A JSON object holding all data for the specified user.
