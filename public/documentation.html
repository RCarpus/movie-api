Movie API Documentation

Author: Ryan Carpus

This API will provide users with access to information about different movies, directors, and genres. Eventually, users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Endpoint Description
/ Displays a short textual message
/documentation.html Displays the API documentation. But you already knew that, didn't you?
/movies Fetches a JSON object containing data about 10 movies
